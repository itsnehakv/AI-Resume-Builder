# :page_facing_up: | AI Resume Builder 
A high-performance, ***full-stack MERN application*** that transforms raw career data into recruiter-ready resumes using **Google Gemini AI via an OpenAI-compatible integration**. This project features professional image processing, real-time design synchronization, and a seamless "split-pane" user experience.

## ✴︎ Features
### ° *UX/UI*
- ***Modern Landing Page***: Features a beautiful hero section, detailed features overview, and professional footer.
- ***Unified Dashboard***: A central hub to "Create" or "Upload" resumes with a persistent "Previous Resumes" gallery.
- ***Interactive Split-Pane***: A seamless left-panel form and right-panel live resume preview for instant visual feedback.
- ***Nested Form Logic***: Add or remove multiple Experience, Education, and Project blocks dynamically.
- ***Privacy Toggle***: Default private mode with a "Share" button and unique URL generation when set to public.

### ° *AI Builder*
- ***Live Preview Engine***: Dynamic split-pane editor with instant visual updates as you type.
- ***Dynamic Enhancement***: Integrated "AI Enhance" buttons in the Professional Summary and Work Experience sections.
- ***AI Text Enhancer***: Professionalizes summaries and work experience using Google Gemini.
- ***AI Photo Studio***: Automated background removal and face-focus via ImageKit.
- ***Custom Design***: Real-time accent color switching and 4 professional resume templates.

### ° *Database & Security*
* **Persistent Storage**: All resumes are securely saved and linked to specific user IDs for future access.
* **Privacy Toggle**: Default private mode with a "Share" button and unique URL generation when set to public.
* **Environment Management**: Securely handles sensitive API keys for Gemini and ImageKit.
  
### ° *Deployment*
- ***Vercel***: Powers the frontend with high-performance edge hosting and seamless CI/CD integration.
- ***Render***: Hosts the backend server, handling API requests, AI processing, and database connections.
- ***Environment Management***: Utilizes secure environment variables to manage sensitive API keys for Gemini and ImageKit.
